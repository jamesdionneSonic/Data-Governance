# Help Center

This is the user-facing help entry point for the Data Governance platform.

If you expose a **Help** option in the application UI, link it to this page first.

This content is now available directly in the app under the **Help Center** sidebar section.

## What do you need help with?

- **Finding data**: start in [User Guide](USER_GUIDE.md)
- **Understanding access and roles**: use [Admin Guide](ADMIN_GUIDE.md)
- **Fixing an error**: open [Troubleshooting FAQ](TROUBLESHOOTING_FAQ.md)
- **Watching the walkthrough**: use [Video Walkthrough Script](VIDEO_WALKTHROUGH_SCRIPT.md)

## Start in the App

1. Go to **Workspace → Command Center** for search-first entry.
2. Continue to **Workspace → Catalog Search** for faceted discovery.
3. Use **Govern → Business Glossary** and **Trust & Compliance** before publishing or sharing.

## Start Here

1. [User Guide](USER_GUIDE.md) — day-to-day usage of search, lineage, discovery, and reporting.
2. [Troubleshooting FAQ](TROUBLESHOOTING_FAQ.md) — fast answers for common issues.
3. [Video Walkthrough Script](VIDEO_WALKTHROUGH_SCRIPT.md) — step-by-step guided demo flow.

## Quick Path

If you are new, follow this order:

1. Read the first section of [User Guide](USER_GUIDE.md).
2. Try one search in the app.
3. Open one object and inspect its lineage.
4. Read [Troubleshooting FAQ](TROUBLESHOOTING_FAQ.md) only if something fails.

## Common User Tasks

- Find an object: see **Discover Trusted Data (Workspace)** in [User Guide](USER_GUIDE.md).
