trigger_sunday

Plain-English Summary

trigger_sunday is an ADF trigger for pl_Marketing_AWS_Export. It matters because it starts the scheduled marketing workflow; if it is stopped or misconfigured, downstream export and mapping data may not refresh on time. Start troubleshooting by confirming trigger state, schedule, and the latest target pipeline run.
